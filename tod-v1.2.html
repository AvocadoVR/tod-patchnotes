Changelog - v1.2

v1.2

New

  • Added support for multiple range placeholders within a single prompt.

Changed

  • You must have 20 non-player placeholders.
    Essential: You must have 20 truths & 20 dares that have no player placeholders. You can use '<local>', '<range=...>' placeholders with them though.

Fixes

  • Fixed overlapping text blocking interaction with the Master lock
  • Fixed range placeholder code issue not properly detecting it.
  • Fixed converting placeholders to return a prompt with its placeholders changed

Removed

  • Removed PTruths and PDares text components from the Custom TOD Sets user interface.
